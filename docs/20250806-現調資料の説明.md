# 現調資料の説明

### assistant  

「現調資料」は「現地調査資料」の略で、現場を実際に訪れて調べた内容をまとめた資料のこと。  

建設や製造、プラント設計などの分野では、現地の寸法、設備配置、配管経路、周辺環境、写真、図面修正点などを記録して、設計や施工計画に反映させるために使う。  

つまり、机上の図面や既存データだけでなく、**実際の現場の状況を確認して得られた情報**をまとめた資料。  

必要なら、現調資料に具体的に何を入れるのかも整理できるけど、どうする？
