# 岩井ファルマテック概要

### assistant  

膜分離とは、**半透膜を使って混合液から特定の成分だけを分離・濃縮・精製する技術**。  

たとえば、水に溶けた不純物を取り除いて**きれいな水だけを通す**ような使い方をする。用途によって「RO（逆浸透）」「UF（限外ろ過）」など種類がある。  

医薬品や食品、上下水処理など幅広く使われる。
